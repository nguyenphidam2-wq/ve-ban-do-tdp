# 🗺️ Vẽ Ranh Giới TDP – Công An Liên Chiểu, Đà Nẵng

Ứng dụng web vẽ và quản lý ranh giới **Tổ dân phố (TDP)** cho Công an phường quận Liên Chiểu, Đà Nẵng.

## ✨ Tính năng

- 🖊️ Vẽ ranh giới TDP trực tiếp trên bản đồ vệ tinh
- 📡 Đồng bộ real-time qua **Firebase Realtime Database**
- 📍 Định vị GPS
- 📥 Xuất dữ liệu dạng **GeoJSON**
- 📐 Tính diện tích tự động
- 📱 Hỗ trợ mobile

## 🛠 Công nghệ

- [Leaflet.js](https://leafletjs.com/) – Bản đồ tương tác
- [Firebase Realtime Database](https://firebase.google.com/) – Lưu trữ & đồng bộ
- HTML / CSS / JavaScript thuần
